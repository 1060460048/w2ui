Adds a column or columns to the grid.
addColumn([before], column)
before string, field name of the column to add before (optional)
column object or array, column(s) to add

Description

Adds a column or columns to the grid by adding items into .columns array and refreshes the grid. The .addColumn() method does not return anything. The first optional argument before is the field name of the column to insert before. If first parameter is not defined, the column will be appended at the end. Second argument column is the column object or an array of column column objects.
