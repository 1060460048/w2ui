Generate HTML template for simple forms.
generateHTML(param)

Arguments

param object, object of parameters

Description

In many cases the HTML template for the form is very typical - fields in one column, may be a few tabs. Instead of creating many HTML files with this typical HTML template, you can generate them. The param argument is the object of the following structure: If you need to have pages or tabs, you can use this structure This method will generate HTML template, assign it to .form_html property and will call .render() method. If you have had form template defined in the element, or in .form_html property, it will be overwritten.
