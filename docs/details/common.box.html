The DOM element where to render the object.
DOMElement, default = null
Usually it is a DIV, but can be any block element. If you change .box property you will need to call .render() method to render widget inside the new DOM element. Usually, there is no need to define .box because it will be automatically set if you specify a selector as it is in the example above.
