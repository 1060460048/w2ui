set(field, obj)
Extends field object with obj object. Returns true on success, false if field does not exist.

Arguments

field string, name of the field
obj object, field object { name, type, required }

Description

The set method will find and update the field object in the field array. If you change the fields array, you will need to call .refresh() method to update it on the screen. Note that this does not change the field value, it changes the fields object, which contains description of the field. In order to change field value you need to update .records array and then call .refresh() method to update it on the screen. After the update the field_1 becomes an integer field. You could have looped through the fields array of the form looking for the field and extending it to achieve same results.
