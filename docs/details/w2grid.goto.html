Opens specified page.
goto(newPage)
newPage int, new page number, 0-based

Description

If the data source is local the .goto() method will update .page property and open specified page rendering records from .records. If data source is remote, the grid will update .page property and retrieve records from .url of the grid. The page numbers are 0-based, which means that goto(0) will open the first page.
