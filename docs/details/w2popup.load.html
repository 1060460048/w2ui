Loads new content of the popup from the server.
$().w2popup('load', options);

Description

The second argument options is a map of properties for the popup (see overview page for valid options). The options argument must have url property defined, which will be called to return popup markup. In many ways this method is similar to the .open method. You can pass other properties in the options argument to the popup. They will be applied on top of properties retrieved from the url. The markup from the server should have sections for title [rel=title], body [rel=body] and buttons [rel=buttons].
When you are loading content from the server, you can optionally add #id at the end of your url. This allows to select a particular popup element by id, if server returns multiple in the same file.
