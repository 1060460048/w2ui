Finds the panel and returns its object.
get(type)
type string, name of the panel

Description

The first argument type is the name of the panel (can be top, left, main, right, preview, bottom). If the panel does not exists in w2layout.panels array, the method will return null. Below is a structure of the panel object: Because JavaScript passes objects by reference, changing the returned object will change the original. You can change the original object with w2layout.set() method.
