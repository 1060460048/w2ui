Number of records per page.
Integer, default = 50
If you have a remote data source, the grid will send limit and offset to the server and server should return the number of records specified in the limit parameter. This parameter is the number of records per page, whereas, limit is calculated depending on the page user currently viewing. If you have a local data source the .recordsPerPage property defines how many records to show per page.
You should set this property during object creation. If you change it at run-time, call the .reload() method to refresh records on the page. It also might be a good idea to set .page property to 0.
