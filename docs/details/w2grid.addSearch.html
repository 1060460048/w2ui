Adds a search or searches to the grid.
addSearch([before], search)
before string, field name of the search to add before (optional)
search object or array, search(s) to add

Description

Adds a search or searhces to the grid by adding items into .searches array. Since the search popup is shown when user clicks the search button, there is no need to refresh the grid. The .addSearch() method does not return anything. The first optional argument before is the field name of the search to insert before. If first parameter is not defined, the search will be appended at the end. Second argument search is the search object or an array of search objects.
