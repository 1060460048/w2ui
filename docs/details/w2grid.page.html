Current page.
Integer, default = 0
The page numbers start with 0. It is not recommended to change the .page property directly, but rather use .goto() method that takes page number as first parameter. There are a number of additional parameters that need to be updated when you move to a new page. If you do change this property directly, call the .reload() method to render correct records on the page. It will do different actions depending on your data source type.
