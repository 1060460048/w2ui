Loads records from specified url.
load(url)
url string, url to pull records from

Description

If you have a local data source and want to load records from url without changing data source to remote you should use the .load() method. If you have used .url property you data source would have been remote and all reload, page navigation, sort, and search actions would have been submitted to this url. This function will do one time server load and will not remember the url.
