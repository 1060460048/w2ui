Called on any server request.
onRequest = function(target, eventData)
You can add event listener during the object creation: or after the object has been created: One of the important features of this event is that you can change the data that will be sent to the server. When the event is emitted, it is setting postData object inside eventData of all the variables that will be sent to the server. If you modify postData, the grid will send modified version to the server. You can also change url property of eventData and the new value will be used
Your event listener can be executed before or after the event has been processed. You can also cancel the default behavior. See Events page in utilities.
