Performs local sort of the records array.
localSort()

Description

Performs local sort of the .records array based on .sortData. Internally, it is used only when the data source is local, however, it can be used in other cases. If the data source is remote and you call this function, it will sort the data in the records array (you still need to call .refresh() method). If user navigates to a new page or otherwise reloads the grid, local sort will not be applied. If your data source is local you do not need to call this function because it is called internally.
