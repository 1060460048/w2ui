Finds and returns matched records.
find(match, [returnRecords])
match object, set of properties to match
returnRecords boolean, if true returns records not their recids

Description

This method returns an array of recids for all matched records. The match argument is a JavaScript object with keys and values. If this keyws and values are the same in the record, it will match it.
