Called to process data that was returned from the server.
requestComplete(status, [cmd], [callBack])
status string, status of the request
cmd string, name of the command (optional)
callBack function, call back function (optional)

Description

This method is called when request returns. First parameters is the status of the request. If it status = 'error' indicated that request. This method will merge returned data to the grid. It uses internal variable w2gird.request_xhr.responseText as the text returned from the server.
