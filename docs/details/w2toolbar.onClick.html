Called when toolbar item is clicked.
onClick = function(target, eventData)
You can add event listener during the object creation: or after the object has been created: You can add a listener to a toolbar item, see .items property. If toolbar item is a menu, then the eventData will have a reference to a menu item user clicked (if any).
Your event listener can be executed before (default) or after the event default behavior has been processed or you can cancel the default behavior. See Events page in utilities.
