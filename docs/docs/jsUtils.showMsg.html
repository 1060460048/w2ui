jsUtils.showMsg()

Displays a message dialog. 

jsUtils.showMsg( text, params )
  text - text of the message
  params - a JavaScript object with additional parameters

Returns HTML div element object

This function will display a message dialog. The text of the message is represended by the first parameter. The second parameter is a JavaScript object with additional parameters. Below is the list of these additional parameters: 

  title - the title of the message
  buttons - html text that will be displayed in the bottom of the dialog, 
    which can be used to output buttons
  width - the width of the dialog in px
  height - the height of the dialog in px

Example

<!DOCTYPE html>
<html>
<head>
    <script src="system/libs/jsUtils.js" type="text/javascript"></script>
    <script type="text/javascript">
        window.onload = function() {
            jsUtils.showMsg('This is my message', { 
                title: 'A Title', 
                buttons: '<input type=button value="close" onclick="jsUtils.hideMsg()">' 
            });
        }
    </script>
<body>
</body>
</html>
