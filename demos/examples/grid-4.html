Local Data

If you define url property of the grid, then all data will be requested from that url. The url will be called on (1) initial grid render, (2) when user navigates to a different page, (3) when user clicks on the column to sort by that column, (4) when user submits search fields, (5) when user clicks the Add New button, (6) when user clicks the Delete button, (7) and some other times when .server method is called.

If the url property is not defined, no calles to the server will be made, but you still can add, remove, update records, search and sort columns, navigate to different pages. This is how to use grid with local data.
