Select/Unselect Records

There are a number of methods that helps you work with the selection. You can select one or multiple record at a time. You can select all records, or clear selection. You can also find out what recorda are currently selected in the grid. If the user navigates to a different page, the selection will be preserved.

As a user you can hold the ctrl key (command key on mac) to select multiple records if the feature is enabled. You can also use shift key to select a range or records.
