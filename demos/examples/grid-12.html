Column Sort

By default column sort is disabled, but it is quite east to enable it. Just add sortable: true property with the column definintion and the column becomes sortable. As a user you can sort on several columns at a time if you cold the ctrl key (command key on a mac).

There are two kinds of sort: local and server side. If the url property is defined for the grid the sort will always be server side. If the url property is not set, it will sort local records array and open the first page.
